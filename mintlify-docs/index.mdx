FinalRun is an AI-powered CLI for testing Android and iOS apps with natural language. You describe a flow the way you'd tell a teammate — "Open the app, tap **Sign in**, enter the email" — and an AI model runs it on a real device, emulator, or simulator, then hands you a pass/fail report with video, screenshots, and device logs. The [AI agent skills](/running/ai-agent-skills) for Claude Code, Cursor, and Codex even write and fix those tests for you. Watch a full run below.
